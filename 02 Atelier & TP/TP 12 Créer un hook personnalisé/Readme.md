# Créer un hook :


Créer un hook useCounter() qui :  
  
initialise un compteur à 0  
  
retourne :
  
count → la valeur actuelle  
  
increment() → ajoute +1  
  
decrement() → enlève -1  
  
Utiliser ce hook dans un composant avec 2 boutons.
